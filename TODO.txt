2. Vérification des formulaires côté client (inscription, connexion, etc.) A FAIRE

3. Edition de profil en direct VVVVVVVVVVVVVVVV

	Les champs sont non modifiables par défaut (grisés).

	Cliquer sur un bouton "modifier" pour activer l’édition d'un champ.

	Bouton "valider" ou "annuler" pour sauvegarder ou annuler sans recharger.

	Quand au moins un champ est modifié, afficher un bouton global "soumettre" pour enregistrer côté serveur.
	
4. Simulation d’attente sur page admin VVVVVVVVVVVVVVVVVV

	Lorsqu’on modifie un paramètre d’utilisateur (ex : bouton VIP), griser le bouton quelques secondes pour simuler un traitement.

5. Trier dynamiquement les résultats de recherche de voyages A PAUFINER

	Ajouter tri par prix, dates, durée, nombre d’étapes, sans rechargement et sans requête serveur.

6. Recalcul dynamique du prix du voyage VVVVVVVVVVVVVVVVVV

	Sur la page de personnalisation du voyage : chaque modification (options, nombre de personnes, etc.) actualise immédiatement le prix affiché en JavaScript.

7. Tout ton code JavaScript doit être dans des fichiers séparés (ex : script.js) et pas directement dans les pages HTML/PHP.

8. Ajout d’une fonctionnalité de "panier" A FAIRE
